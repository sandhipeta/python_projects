# Student Marks Calculator (Tkinter GUI)

##  Project Description

This is a simple **GUI (Graphical User Interface)** application built using **Python Tkinter**.

The application allows the user to:

* Enter student name
* Enter marks for 3 subjects (Math, Science, English)
* Calculate total marks
* Calculate average marks
* Display Pass or Fail result

The result is displayed instantly in the application window.

##  How It Works

1. User enters:

   * Student Name
   * Math marks
   * Science marks
   * English marks

2. When user clicks **Calculate Result** button:

   * Total marks are calculated
   * Average marks are calculated
   * Result shows Pass or Fail

3. Pass condition:

   * Average marks >= 35 → Pass
   * Average marks < 35 → Fail
