# BMI Calculator (Tkinter)

## Project Description

This is a simple **BMI (Body Mass Index) Calculator** built using **Python Tkinter GUI**.
The application allows users to enter their **height (in centimeters)** and **weight (in kilograms)** and calculates their BMI value.
Based on the BMI result, the program shows the **health category** such as underweight, healthy, or overweight.

This project is useful for beginners learning:

* Python GUI programming
* Tkinter widgets
* Basic calculations
* Conditional statements

---

## Technologies Used

* Python
* Tkinter (GUI library)

---

## Features

* Simple and user-friendly interface
* Takes user input for height and weight
* Calculates BMI using formula
* Displays health category result
* Beginner-friendly project

---

## BMI Formula Used

BMI is calculated using the formula:

BMI = weight (kg) / height (m²)

Height is converted from centimeters to meters before calculation.

---

## BMI Categories

| BMI Range    | Category             |
| ------------ | -------------------- |
| Less than 16 | Severely Underweight |
| 16 – 18.5    | Underweight          |
| 18.5 – 25    | Healthy              |
| 25 – 30      | Overweight           |
| Above 30     | Severely Overweight  |

## How to Run the Project

### Step 1: Install Python

Make sure Python is installed on your system.

### Step 2: Run the file

Open terminal or command prompt and run:

python bmi_calculator.py

---

## Application Workflow

1. User enters height in centimeters
2. User enters weight in kilograms
3. Click on "Calculate BMI" button
4. BMI result and health status will be displayed
